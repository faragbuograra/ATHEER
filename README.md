# Amyal Company Landing Page

A modern, responsive, corporate landing page for Amyal Company (Libya).

## Tech Stack
- React
- Vite
- Tailwind CSS
- Three.js / R3F (for 3D Hero)
- Framer Motion

## Features
- Bilingual (Arabic/English) with RTL support
- Dark Mode Themed (Blue & Dark Slate)
- 3D Hero Element
- Responsive Design

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
